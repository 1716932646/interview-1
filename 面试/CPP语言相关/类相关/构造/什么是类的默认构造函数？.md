默认构造函数：未提供任何实参，来控制默认初始化过程的构造函数称为默认构造函数。


```
#include <iostream>

using namespace std;

class A
{
public:
    A(){ // 类的默认构造函数
        var = 10;
        c = 'q';
    }
    int var;
    char c;
};

int main()
{
    A ex;
    cout << ex.c << endl << ex.var << endl;
    return 0;
}
/*
运行结果：
q
10
*/
```

说明：上述程序中定义变量 ex 时，未提供任何实参，程序运行时会调用默认的构造函数。
